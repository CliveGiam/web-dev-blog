Week 4 Dev Blog

Responsive Images

What I have learned this week:

  • Responsive images will automatically resize based on screen size.
  • To create a responsive image, relative units such as % or em are used so that the image will resize based on the context around it.
  • Images can be forced to not scale beyond a certain size using max-width and width.
  • 'srcset' allows images of multiple resolutions to be set to load at certain screen sizes.

Week 7 Dev Blog

DOM, Event Handling and Local Storage

What I have learned this week:

  • DOM
  • A DOM Tree is created when a browser loads a webpage which is serves as a model of the page.
  • The DOM Tree is made up of nodes which represent elements of the page and can be accessed, queried and updated.
  • Event handling
  • When users interact with websites, events occur and can be used to run specific scripts that alter the page.
  • Local storage
  • Data can be stored in the user's browser using 'localStorage' and 'sessionStorage' objects
  • Data stored using local storage is retained even when the tab is closed and allows all windows and tabs to access it
  • Session storage does not retain data after tab closure and only specific windows or tabs are able to access it
  • Size of data stored using storage objects is limited to 5mb
End of week 7.
